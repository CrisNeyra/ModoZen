# Assets de Modo Zen

Coloca aquí tus archivos multimedia:

1. **logo.png** - Imagen para la pantalla de login (reemplaza el emoji 🧘)
2. **background.mp4** - Video de fondo para la pantalla principal

## Formatos recomendados

- **Imagen**: PNG o JPG (recomendado 300x300px para el logo)
- **Video**: MP4 (H.264 codec, recomendado 1080p o menor para mejor rendimiento)
