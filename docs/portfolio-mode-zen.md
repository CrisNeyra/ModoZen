# Modo Zen para portfolio

## Texto corto

Modo Zen es una app mobile de meditacion desarrollada con React Native. Permite iniciar sesiones guiadas, reproducir musica ambiente, registrar el estado de animo antes y despues de cada practica y cerrar la experiencia con feedback del usuario.

## Texto medio

Desarrolle Modo Zen como una app mobile enfocada en bienestar y experiencia de usuario. La aplicacion incluye autenticacion, sesiones de meditacion guiadas, audio ambiente, seguimiento del estado de animo y una instancia de feedback al finalizar la practica. El proyecto fue construido con React Native y TypeScript, con navegacion nativa, persistencia local y una interfaz pensada para transmitir calma, claridad y continuidad de uso.

## Stack

- React Native
- TypeScript
- React Navigation
- AsyncStorage
- Android build con Gradle

## Que mostrar en el portfolio

1. Una portada con nombre, rol y objetivo del producto.
2. Un video corto real en celular de 30 a 60 segundos.
3. Cuatro o cinco capturas de pantallas clave.
4. Un bloque tecnico con stack, decisiones y desafios.
5. Un boton para descargar la APK si queres que puedan probarla.

## Flujo ideal para la demo

1. Login.
2. Home principal.
3. Inicio de una meditacion.
4. Desarrollo de una sesion.
5. Pantalla de cierre.
6. Modal de feedback.
